== Description ==

Plugin Name is a WordPress plugin that provides a simple and lightweight solution for creating a marquee effect with text content on your website. The plugin emulates the marquee functionality using JavaScript and CSS animation.

== Features ==

- Creates a marquee effect for text content.
- Smooth and continuous scrolling animation.
- Responsive design, adapts to different screen sizes.
- Customizable animation speed.
- Works well with any WordPress theme.

== Installation ==

1. Upload the `plugin-name` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add the CSS class `marquee_block` to the HTML element that you want to apply the marquee effect to.
4. Optionally, add the `data-marquee-speed` attribute to the HTML element to customize the animation speed (in milliseconds). If not provided, the default speed of 200 milliseconds will be used.

== Frequently Asked Questions ==

= How do I customize the animation speed? =

You can customize the animation speed by adding the `data-marquee-speed` attribute to the HTML element that has the `marquee_block` class. Set the value of the attribute to the desired animation speed in milliseconds.

For example:
<div class="marquee_block" data-marquee-speed="300">Lorem ipsum dolor sit amet</div>
```
